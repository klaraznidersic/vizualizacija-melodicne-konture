# Vizualizacija melodične konture

Pred uporabo se prepričajte, da je v vašem okolju nameščeno:
* **Python 3.x**
* Potrebne zunanje knjižnice (navedene v `requirements.txt`)


1. **Klonirajte repozitorij:**
   ```
   git clone [https://github.com/klaraznidersic/vizualizacija-melodicne-konture.git](https://github.com/klaraznidersic/vizualizacija-melodicne-konture.git))
   cd vizualizacija-melodicne-konture
   ```
   
2. **Namestite potrebne pakete**
   (priporočena uporaba virtualnega okolja)
   ```
   pip install -r requirements
   ```
4. **Uporaba**:
   
      Za izbrano mapo:
      ```
      python3 generator.py <pot_do_mape_z_musicxml>
      ```
      Za datoteke v mapi Testna_mapa:
      ```
      python3 generator.py
      ```
